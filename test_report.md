# Отчёт о проверке

## Автоматические проверки

Проверено 18.07.2026 на Python 3.12.10, Docker 29.6.1 и локальной Ollama с моделью `qwen2.5-coder:7b`.

| Проверка | Результат |
| --- | --- |
| `python -m unittest discover -s tests -v` | 21 из 21 теста пройдено |
| `python evals/run_eval.py` | 5 из 5 контрактных сценариев, критических ошибок 0 |
| `python evals/run_shopping_eval.py` | 3 из 3 реальных LLM-сценариев, критических ошибок 0 |
| `ruff check .` | замечаний нет |
| `python -m compileall -q ...` | все модули компилируются |
| `docker build -t aiprod-llm-service:local .` | образ собран |
| HTTP-проверка контейнера | `/health` вернул 200, первый `/chat` вернул 200, повторный запрос пришёл из кеша |
| Визуальная проверка Swagger | четыре читаемых скриншота: 200, cache hit, 422 и 503 |
| Визуальная проверка Streamlit | десктоп 1280 px, мобильный экран 390 px, успех и ошибка валидации |

Контрактный eval привязан к dataset `aiprod-service-contract-v1`, схеме `chat-api-v1` и версии evaluator `contract-eval-v1`. SHA-256 dataset: `a85821c7f5b4bb76aec0b1fb9648ea739526ecb45a5aac7c75f3b3a5ce5de4e4`.

Eval интерфейса привязан к dataset `shopping-ui-golden-v1`, схеме `shopping-ui-text-v1`, evaluator `shopping-eval-v1`, модели `qwen2.5-coder:7b` и температуре `0.2`. SHA-256 dataset: `cb212ac48b812f07bee236e2c04fd700afff701d57b12cb55c2349e0017e01ec`; prompt: `bff011ba5759c180ab2dee565d3f2a55dad86a42dcc3ccd8b884b52c4c6ae640`. Латентность трёх реальных случаев: 3258–18806 мс, стоимость локальной Ollama — 0. Пользователь остаётся финальным проверяющим состава покупок перед использованием результата.

Реальный запрос из Docker-контейнера к Ollama занял 3312 мс. Идентичный повторный запрос вернулся за 0 мс с `cached=true`.

## Ручной чек-лист

- [x] Корректный запрос возвращает HTTP 200 и непустой `answer`.
- [x] Пустой `message` возвращает понятную ошибку HTTP 422.
- [x] `message` длиннее 1000 символов возвращает HTTP 422.
- [x] Временный сбой LLM повторяется ограниченное число раз.
- [x] Недоступная LLM приводит к контролируемому HTTP 503, сервис не падает.
- [x] Повторный идентичный запрос возвращается из кеша с `cached=true`.
- [x] Изменение модели, температуры или системного prompt меняет ключ кеша.
- [x] Логи содержат `request_id`, `cache_hit/cache_miss` и длительность.
- [x] Секреты и необработанный пользовательский текст не попадают в логи по умолчанию.
- [x] Streamlit показывает поле ввода, два списка, кнопку и область результата.
- [x] Пустой и длинный ввод не вызывают API и дают короткое сообщение.
- [x] Ошибки доступа, сети, провайдера и пустого ответа не раскрывают traceback или секреты.
- [x] Реальный запрос через Streamlit к локальной Ollama возвращает список покупок.
- [x] На ширине 390 px горизонтальной прокрутки нет.

## Остаточный риск

`FastAPI TestClient` выводит предупреждение о будущей миграции Starlette с `httpx` на `httpx2`. На текущую версию и результаты тестов это не влияет. HTTP-проверка собранного Docker-контейнера пройдена отдельно.
