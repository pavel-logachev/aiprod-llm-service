# Состояния экрана

| Состояние | Что видит пользователь | Доступное действие | Критерий |
| --- | --- | --- | --- |
| По умолчанию | Пустая форма и примеры | Заполнить и отправить | Все поля имеют видимые подписи |
| Пустой ввод | Короткая ошибка | Исправить текст | API не вызван |
| Длинный ввод | Короткая ошибка | Сократить текст | API не вызван |
| Загрузка | Индикатор «Собираем список…» | Дождаться | Нет traceback и двойной отправки |
| Успех | «Список готов» и результат | Изменить форму и повторить | Результат читаем на 360–1440 px |
| Нет доступа | «Не настроен доступ к сервису» | Настроить окружение | Нет секрета или URL |
| Сеть | Предложение повторить | Повторить | Одна короткая фраза |
| Провайдер недоступен | Предложение попробовать позже | Повторить позже | Одна короткая фраза |
| Пустой ответ | Предложение переформулировать | Изменить блюдо | Нет сырого ответа |

Клавиатура проходит по полю, спискам и кнопке в визуальном порядке. На узком экране элементы Streamlit перестраиваются вертикально, а длинный русский текст переносится без горизонтальной прокрутки.
